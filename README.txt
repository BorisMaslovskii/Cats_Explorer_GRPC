Пример моей практики с HTTP API и GRPC
Сперва было сделано HTTP API по доступу к базе котиков. Структура Cat простая - id, name, age, color. Сделал обработчики для get-select, put-insert, post-update, delete-delete. Потом переделал все под GRPC. Cерверную часть GRPC запускал в докере, он обращался через user-defined bridge network к контейнеру Postgres с тестовой базой котиков. Тестовая клиентская часть была создана для проверки работоспособности.
